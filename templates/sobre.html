Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Laboratório de Programação. A aplicação utiliza Flask como framework web e Bootstrap para o design. Os templates HTML são construídos com Jinja, que acompanha o Flask.

Tecnologias utilizadas:

  • Flask - Framework web
  • Bootstrap - Para o design
  • Jinja - Templates
  • HTML/CSS - Interface
